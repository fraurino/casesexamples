```sql
CREATE TABLE ANEXO_REDUCAO (
    ID     INTEGER NOT NULL PRIMARY KEY,
    ANEXO  VARCHAR(4),
    NCM    VARCHAR(8),
    NBS    VARCHAR(8),
    PRED   NUMERIC(5,2)
);
```

--
- `ID`: Identificador único do registro.
- `ANEXO`: Número do anexo conforme LC 204.
- `NCM`: Código NCM do produto (quando aplicável).
- `NBS`: Código NBS do serviço (quando aplicável).
- `PRED`: Percentual de redução da alíquota.

Essa tabela armazena as reduções de alíquota conforme previsto nos anexos da legislação.
